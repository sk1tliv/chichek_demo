# Chichek - Herbal Tea

![Chichek Web Sitesi Mockup](mockup.jpg)

##  Proje Hakkında

Bu proje, **Chichek** adlı bitkisel çay kuruluşunun ürünlerini ve markasını dijital ortamda sergilemek amacıyla geliştirilmiş bir tanıtım web sitesidir. Amacımız, kullanıcı dostu bir arayüz ile en kaliteli doğal bitki çayları hakkında detaylı bilgi sunmaktır.

##  Canlı Gösterim

Sitenin yayınlanmış haline aşağıdaki bağlantıdan ulaşabilirsiniz :

* **www.chichektea.com**

## Kullanılan Teknolojiler

* **HTML5:** Sitenin yapısal iskeleti için.
* **CSS3:** Temel stil ve görsel düzenlemeler için.
* **Bootstrap:** Hızlı ve duyarlı tasarım geliştirmeyi sağlayan popüler CSS çatısı.
* **Font Awesome:** Site içerisinde kullanılan ikonlar ve görsel öğeler için.
* **JavaScript :** Arayüz etkileşimleri ve dinamik öğeler için.
